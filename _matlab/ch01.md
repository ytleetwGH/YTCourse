# CH01 矩陣運算
這是 MATLAB 的第一章。
MATLAB 矩陣運算涵蓋線性代數的標準運算（如 
+, -, *, /, ^）和逐元素操作（如 .*, ./, .^），並區分「矩陣運算」和「陣列（元素級）運算」；標準矩陣運算遵循線性代數規則（例如 A*B），而加點的運算符則對應元素操作（例如 A.*B），對於矩陣處理和解線性方程組至關關重要，可使用轉置 A'、行列式 det(A)、求逆 inv(A)、左除 A\B（解 
 
Ax=Bcap A x equals cap B
𝐴𝑥=𝐵
）等功能，方便高效地進行數值計算。 
1. 基本算術運算符 
•	加/減: A + B, A - B (元素對應相加減)
•	乘法:
o	A * B: 矩陣乘法（滿足線性代數規則）
o	A .* B: 元素對應相乘 (點乘)
•	除法:
o	A / B: 矩陣右除 (求解 
 
Ax=Bcap A x equals cap B
𝐴𝑥=𝐵
中的 
 
x=A-1Bx equals cap A to the negative 1 power cap B
𝑥=𝐴−1𝐵
)
o	A \ B: 矩陣左除 (求解 
 
Ax=Bcap A x equals cap B
𝐴𝑥=𝐵
中的 
 
x=A-1Bx equals cap A to the negative 1 power cap B
𝑥=𝐴−1𝐵
)
o	A ./ B: 元素對應相除 (點除)
•	次方:
o	A ^ n: 矩陣自乘 (例如 A^2 = A*A)
o	A .^ n (或 k .^ A): 元素對應求冪 
2. 常用矩陣函數 
•	轉置: A' (共軛轉置) 或 A.' (非共軛轉置)
•	行列式: det(A)
•	求逆: inv(A) (需方陣且可逆)
•	尺寸變換: reshape(A, m, n)
•	求和/積: sum(A) (按列), sum(A, 2) (按行), prod(A)
•	查找元素: find(A) (返回非零元素位置)
•	範數/秩: norm(A) (2-範數), rank(A) 
3. 區分矩陣與陣列操作 
•	* (矩陣乘法) vs .* (點乘): 必須注意維度。A*B要求A的列數等於B的行數；A.*B要求A和B大小相同。
•	^ (矩陣自乘) vs .^ (點求冪): A^2是矩陣A乘以自身；A.^2是A中每個元素平方。
•	\ (左除) vs ./ (點除): A\B是解線性方程組

𝐴𝑥=𝐵
；A./B是元素對應相除。
