### 進階：如何新增圖片到網頁？
在 GitHub 儲存庫頁面，點擊 Add file > Upload files。

上傳你的圖片檔（例如 my_photo.jpg）。

在你的 .md 檔案中加入這行： ![圖片說明](./my_photo.jpg)
